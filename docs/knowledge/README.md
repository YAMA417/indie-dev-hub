# ナレッジベース

プロジェクト横断で再利用可能な技術知見をテーマ別に整理する。

## カテゴリ一覧

| ファイル | テーマ | 概要 |
|---------|-------|------|
| <!-- [nextjs.md](./nextjs.md) --> | Next.js | App Router、RSC、最適化など |
| <!-- [authentication.md](./authentication.md) --> | 認証 | Clerk、Auth.js、セッション管理 |
| <!-- [deployment.md](./deployment.md) --> | デプロイ・インフラ | Vercel、Cloudflare Workers、CI/CD |

必要に応じてカテゴリファイルを追加してください。

## 知見の追加方法

`/knowledge-add` コマンドを使用するか、直接ファイルを編集する。

### フォーマット

```markdown
### タイトル（YYYY-MM-DD）

**状況**: どんな場面で遭遇したか
**解決策/知見**: 学んだこと
**関連プロジェクト**: 該当プロジェクト名（あれば）
```
